Tablas

Una tabla permite la distribución la información organizada por filas(rows) y columnas(cols). Se utiliza la etiqueta html de table. Ejemplo:

Lunes Martes Miércoles Jueves Viernes
FH BD ISO ISO ISO

Problemas

Vamos a experimentar problemas cuando el nº de columnas no sea el mismo en todas las filas. Lo siguiente está mal... Ejemplo:

Lunes Martes Miércoles
FH BD ISO ISO ISO

Soluciones

Si deseamos que una celda ocupe más de una columna o fila entonces utilizaremos las propiedades colspan o rowspan. Ejemplo:

Lunes Martes Miércoles
FH BD ISO ISO ISO

Otro ejemplo con una celda que ocupa dos filas. Ejemplo:

Lunes Martes Miércoles Jueves Viernes
BD ISO ISO ISO
