¿Cuando conviene utilizar los distintos tipos de posicionamiento?
- El posicionamiento static simplemente es el que tiene por defecto las cajas, así que si no queremos usar ninguno o no es necesario, este será el más indicado.
- El posicionamiento relative nos puede servir para indicar el sitio que normalmente va tener una imagen en el sitio web, para crear una cabecera con una imagen nos puede servir.
- El posicionamiento fixed nos podría servir para manetener un logo siempre a la vista.
- El posicionamiento absolute nos permite realmente realizar lo mismo que con el relativo, pero por todo el espacio del sitio web, nos puede permitir crear combinaciones de estilo
en un solo espacio asignado mediante contenedores.
- El posicionamiento sticky nos permite mantener una caja en el scroll de pantalla a la vista del usuario en la misma posición que donde estaba, este posicionamiento nos puede servir
para publicidad.

¿Para que sirve la propiedad clear en el posicionamiento flotante?
- La propiedad clear nos permite definir que sucede con los elementos que están cerca de un elemento con la propiedad float.

¿Podemos combinar en una caja posicionamiento flotante y relativo? -Sí.
¿Tiene sentido utilizar top y left cuando tenemos posicionamiento flotante? -No, ya que tenemos setencias especiales para ello.

¿Qué ocurre si tenemos texto alrededor de un elemento posicionado de manera flotante?
- Que podemos añadirle la propiedad clear para que forme parte del contenedor, o que automáticamente se vaya para un lado.
